## Quick Start (30 seconds)

### 1. View Your Docs

Open terminal in your docs folder and run:

```bash
npx nlddoc serve .
```

That's it! Your browser opens with a beautiful site.

### 2. Make Changes

Edit any `.md` file and save. Your browser updates automatically. No refresh needed.

### 3. Share It (Optional)

When you're ready to share:

```bash
npx nlddoc build . ./website
```

This creates a `website` folder you can upload anywhere.
